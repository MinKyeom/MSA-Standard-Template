# 배포 등록 양식

**push 직후 CI를 바로 돌리려면** [Nginx-운영-가이드.md §1](./Nginx-운영-가이드.md#1-git-push-직후-cicd를-바로-시도할-수-있는지) 체크리스트를 함께 보세요.

| 용도 | 파일 / 등록 위치 |
|------|------------------|
| GitHub → Actions → **Secrets** `HOST`, `USERNAME`, `KEY`, (`DEPLOY_PATH` 선택) | [templates/github-secrets.txt](./templates/github-secrets.txt) |
| GitHub → Actions → **Secrets** `ENV_VARS` | 로컬 `.env` **전체**를 한 Secret(다줄)로 붙여넣기 |
| GitHub → Actions → **Secrets** `SSH_PRIVATE_KEY` | 사용 안 함 → **`KEY`** 로 통일 |
| 서버 초기 1회 | `DEPLOY_PATH` 가 비면 `/home/<USERNAME>/<저장소이름>` 에 **같은 Git 저장소 clone** |

CI 가 `ENV_VARS` 끝에 `DOCKER_IMAGE_PREFIX`(ghcr)·`IMAGE_TAG`(커밋 SHA) 를 추가합니다.
